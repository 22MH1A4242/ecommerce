# Ecommerce Project

This is a PHP-based eCommerce website built with XAMPP (Apache + MySQL).

## Features

- User Registration & Login
- Product Listings
- Shopping Cart
- Order Management

## Setup

1. Clone the repo
2. Import the database from `/database/yourfile.sql`
3. Run on XAMPP server: `http://localhost/ecommerce`
